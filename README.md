# Westlake Test App - GitHub Only

This is a test/sandbox version.

No Supabase.
No login.
No API keys.
No SQL.

It stores data locally in the browser using localStorage.

Use this to test:
- Home dashboard
- Recent customers
- Recent invoices
- Upcoming jobs
- Customers with billing/property address
- Customer history
- Invoices and estimates
- Estimate signature
- Estimate to job
- Job to invoice
- Reports
- Revenue by month
- Revenue by service
- Revenue by category
- Default invoice terms
- Payment instructions

Upload all files to a GitHub Pages repository.
